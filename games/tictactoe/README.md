# Tic-Tac-Toe Lab

A standalone Tic-Tac-Toe web app with the shared game style system.

## Features

- Player vs Player and Player vs AI
- AI levels: easy, medium, hard
- Theme selector with six themes and persistence
- Score tracking
- PWA manifest and service worker

## Run Locally

Open `index.html` with a static server from the repository root.

Example:

```bash
python3 -m http.server 5500
```

Then visit:

`http://localhost:5500/games/tictactoe/`
