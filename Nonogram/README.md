# Nonogram Lab

A standalone Nonogram web app with shared game styling and theme support.

## Features

- 5x5 and 10x10 puzzle sets
- Fill / mark / erase modes
- Clue rendering for rows and columns
- Check and hint actions
- Theme selector with persistence
- PWA manifest and service worker

## Run Locally

Start a static server from repository root:

```bash
python3 -m http.server 5500
```

Then visit:

`http://localhost:5500/Nonogram/`
